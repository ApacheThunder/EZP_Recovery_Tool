====== NDS Backup Tool Slot2 V0.34 ======

Backup/restore of NDS ROM SAVE data using Magicon for Slot-2
It is also a tool for backing up ROM images.
This may cause problems such as damage to the NDS ROM SAVE data.
Please use at your own risk.


<Introduction method>
"NDS_Backup_Tool_Slot2.nds" is DLDI compatible, so please use it according to the magic controller you are using.
Please perform patching, etc. By default, it is not installed.

- Modify "NDS_Backup_Tool_Slot2.ini" using an editor etc. according to the usage environment.
·Copy "NDS_Backup_Tool_Slot2.nds(bin)" and "NDS_Backup_Tool_Slot2.ini" to the root of the SD.
If "NDS_Backup_Tool_Slot2.ini" is not present, it will operate with the default value (usually it is okay without it).

<NDS_Backup_Tool_Slot2.ini>
Enter various operating environment settings. The characters on the line after '#' and '!' are treated as comments.
·FileDir
Specify the SD directory that manages backed up files (initial value is /NDS_Backup).
·SaveFile 0
Specify the size of the SAV file when backing up ROM SAVE. of the size of the specified number
The unit is KByte. If you specify 0, it will be the ROM SAVE size (initial value is 0).
Please specify 512 for R4/M3S and 520 for DSLink.
·Trim
Specify this to trim unused areas when backing up NDS ROM (default value is no trim).


<How to use>
-Start "NDS_Backup_Tool_Slot2".
·Use <L><R> to switch between Save Backup, Save Restore, and Rom Backup.

(Save Backup)
·<A>: Updates the contents of the specified SAV file as a backup with the SAVE contents of the ROM.
·<B>: Create a new SAV file and back up the SAVE contents of the ROM.
     The name of the automatically created SAV file is GameTitle_Gamecode_No.sav.
     No is automatically calculated from 00 to 99.

(Save Restore)
·<A>: Restores (writes) the contents of the specified SAV file to SAVE in the ROM.
·<B>: Initialize the ROM SAVE area with 0xFF.

(Rom Backup)
·<A>: Updates the contents of the specified NDS file as a backup with the dumped contents of the ROM.
·<B>: Create a new NDS file and back up the dumped contents of the ROM.
     The name of the automatically created NDS file is GameTitle_GamecodeRomVer.nds.
If a file with the same name already exists, it will be automatically overwritten.

 (common)
·<X>: Replace the NDS ROM in Slot-1.
·<START>: Exit the program and turn off the NDS.



<Caution>
- With DLDI for Slot-2, file creation and writing may be unstable.
Be sure to back up the SD before using it.
- If the SAVE type of the ROM cannot be determined, "Unknown" will be displayed. It is displayed as Unknown, or it is clearly
Some ROMs display different SAVE types or some ROMs display a ROM set message no matter how many times they are set.
Please contact us if you have any.
-Depending on the ROM card, it may not be possible to recognize it correctly the first time (it may be possible to recognize it by inserting and removing it several times).
For Macronix 1024Mbit (128MB) etc., please use Deitel's "recognition improvement adapter" or adjust the terminals on the board.
